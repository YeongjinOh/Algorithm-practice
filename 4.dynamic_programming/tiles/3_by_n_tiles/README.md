#No.2133 &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;3xn tiles

##Problem

2×N 크기의 벽을 2×1(그리고 1×2) 크기의 타일로 채우는 경우의 수를 구하는 문제는 쉽다. 그렇다면 3×N 크기의 벽을 2×1, 1×2 크기의 타일로 채우는 경우의 수를 구해보자.  

##Input

첫째 줄에 N(1≤N≤30)이 주어진다.  

##Output

첫째 줄에 경우의 수를 출력한다.  

##Example

####input

2  

####output

3  

##Limitation

Time : 2(s)  
Memory : 128MB
